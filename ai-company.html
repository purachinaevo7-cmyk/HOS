E. 操作ガイド
  1. Task JSONを作成し、コピーまたはダウンロード。
  2. mockは python -m orchestrator.cli run tasks/inbox/investment_analysis.sample.json --executor mock。
  3. GitHub Actionsは HOS AI Company workflow_dispatch の task_json に貼付。
  4. Artifact hos-run-<run_id> を取得し、Run Viewerへ貼付。
  5. outputs/investment_commander.json をInvestment Commanderの取り込み欄へ貼付。
  6. Gemini無料枠はGEMINI_API_KEYが必要です。無料枠は保証されず、上限到達時はpartial/failedになりOpenAI APIやmockへ自動切替しません。OpenAI APIは別途API契約が必要です。
UIから直接実行できない操作は「実行済み」と表示しません。
